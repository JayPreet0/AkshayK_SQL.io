Performing Arithmatic Operations


To perform arithmetic operations in SQL, use the following syntax:
SELECT Value1 (ArithmeticOperator) Value2;


Example:
Let's consider the multiplication operation:

SELECT 2 * 3;

This operation will give an output of 6.
Now, let's take another example where we utilize the pow function in SQL to calculate the power of a number.

SELECT pow(a, b);

In this function, "a" represents the base number, and "b" represents the exponent.


Example:

SELECT pow(2, 3);

This operation will give an output of 8.

Performing Comparison Operations


To perform comparison operations in SQL, use the following syntax:

SELECT Value1 (Comparison Operator) Value2;

Please note that the output of these operations will always be in binary format,i.e., 0 and 1.
Where, "0" stands for False and "1" stands for True.


Example:

SELECT 2 > 3;

This operation will output 0, i.e., False.
