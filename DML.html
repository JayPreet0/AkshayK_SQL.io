Inserting record in a table


Insert into Table_Name
            Values (Value 1, Value 2, Value 3,......Value n);


Example:
If there is a table who's syntax is 


Create Table Student (
                                         ID int, 
                                         Name varchar (50),
                                         Address varchar (100),
                                         City (15)
                                         );
and now, if we have to insert a record in the table, 
Our syntax will be:


Insert into Student
			Values(97, "Akshay", "LinkedIN", "Pune")


We always have to remember, 
Maintaining the order type and input data type of values 
is utmost important.

Updating a record in a table


Update Table_Name
            Set (
	 Column 1 = Value 1,
		 Column 2 = Value 2,
		 .
		 .
		 .
		 Column n = Value n
				)
	    where condition/s;


Example:
Update Student
		   Set (ID = 1)
		   where Name = "Akshay";

Deleting a record in a table


Delete from Table_Name
where condition/s;


Example:
Delete from Student
where Name = "Akshay" and ID=1;
